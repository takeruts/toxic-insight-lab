# 🧪 TOXIC INSIGHT LAB (毒吐き心理分析ラボ)

あなたの心に溜まった「澱（おり）」や「毒」を抽出し、深層心理を分析する実験的チャットプラットフォームです。

## 🧪 About the Lab
このラボでは、二人の専門家（Dr.ナギ / Dr.リョウ）があなたの言葉を待っています。
単なる励ましや共感ではありません。ユング、アドラー、フロイトといった心理学の多角的な視点から、あなたが自分でも気づいていない「真実の肖像」を浮き彫りにします。

## ✨ Features
- **抽出フェーズ**: 3回〜の対話を通じて、感情の成分を細かくサンプリングします。
- **知的分析**: 孤独や寂しさといった安易な結論に逃げない、鋭い心理学的考察。
- **最終結晶化**: 蓄積された成分から、500文字程度の濃密な「真実の肖像」を生成します。
- **対話履歴**: 自分が吐き出した毒が浄化されていく過程を、タイムライン形式で遡ることができます。

## 🤖 Powered by AI
このプロジェクトは、**Google Gemini API (Gemini 1.5 Flash)** の高度な言語モデルを活用して構築されています。
AIによる多層的な分析を導入することで、人間の精神の複雑さにアプローチする新しいユーザー体験を追求しています。
また、本プロジェクトの設計・開発プロセスにおいてもAIとの共創（AI-human collaboration）が行われています。

## 🛠 Tech Stack
- **Framework**: Next.js 14 (App Router)
- **AI Engine**: Google Gemini API
- **Styling**: Tailwind CSS
- **Animation**: Framer Motion
- **Language**: TypeScript

## 🚀 Getting Started

### 1. リポジトリをクローン
```bash
git clone [https://github.com/あなたのユーザー名/toxic-insight-lab.git](https://github.com/あなたのユーザー名/toxic-insight-lab.git)
cd toxic-insight-lab
```

### 2. 依存関係のインストール

```bash
npm install
```

### 3. 環境変数の設定

プロジェクトのルートに `.env.local` ファイルを作成し、Gemini APIキーを設定してください。

```env
GOOGLE_GENERATIVE_AI_API_KEY=あなたのAPIキー
```

### 4. 開発サーバーの起動

```bash
npm run dev
```
